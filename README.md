
# Project Title

A brief description of what this project does and who it's for

This is a frame Work designed using Play Wright with JS with POM
## Run Locally

Clone the project

```bash
  git clone git@github.com:Pract-devops/Playwright-FrameWork-POM.git
```

Go to the project directory

```bash
  cd Playwright-FrameWork-POM
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
